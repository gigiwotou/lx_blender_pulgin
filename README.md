# 流星蝴蝶剑 Blender 工具箱

将原版3dsmax流星蝴蝶剑辅助工具移植到Blender的插件。

## 功能列表

### 导入功能
- **SKC人物模型导入** - 支持导入SKC格式的人物模型（需要配套BNC骨骼文件）
- **GMB/GMC模型导入** - 支持导入GMB/GMC格式的模型文件
- **AMB动作导入** - 支持导入AMB格式的人物动作文件

### 导出功能
- **GMB模型导出** - 导出流星格式的模型文件
- **DES场景文件导出** - 导出场景对象位置信息
- **COB碰撞文件导出** - 导出碰撞检测数据
- **FMC动画导出** - 导出道具/武器动画
- **POS姿态文件导出** - 导出姿态信息
- **AMB动作导出** - 导出人物动作
- **WP路径点导出** - 导出导航路径点

## 安装方法

1. 打开Blender（3.0及以上版本）
2. 进入 `编辑 > 首选项 > 插件`
3. 点击 `安装...` 按钮
4. 选择 `lx_blender_plugin` 文件夹
5. 勾选 `流星蝴蝶剑 Blender 工具箱` 插件
6. 重启Blender

## 使用方法

### 导入模型
1. 在3D视图中打开侧边栏
2. 切换到 `流星蝴蝶剑` 选项卡
3. 点击 `导入SKC模型` 或 `导入GMB/GMC模型`
4. 选择要导入的文件

### 导出模型
1. 在Blender中选择要导出的对象
2. 在侧边栏的 `流星蝴蝶剑` 选项卡中
3. 点击相应的导出按钮
4. 选择保存路径

## 文件格式说明

| 格式 | 用途 |
|------|------|
| SKC | 人物模型 |
| BNC | 人物骨骼 |
| GMC | 文本格式模型 |
| GMB | 二进制格式模型 |
| DES | 场景描述文件 |
| COB | 碰撞数据文件 |
| AMB | 人物动作文件 |
| FMC | 道具动画文件 |
| POS | 姿态文件 |
| WP | 路径点文件 |
